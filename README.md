# 🍦 Froyo & Yogurt Paradise - CT188 Web Lab Project

A modern, responsive, and visually appealing static website for a premium Frozen Yogurt & Ice Cream shop. This project was built as part of the **CT188 - Introduction to Web** course lab assignment.

🌐 **Live Demo:** [Click here to view the live website](https://williamtheduck.github.io/Web-Lab-Exercises/)

---

## ✨ Features

* 📱 **Fully Responsive:** Optimized for mobile, tablet, and desktop screens.
* 🎨 **Modern UI/UX:** Clean dessert-themed design with soft colors (Pink/Yogurt tones).
* 🛒 **Product Catalog:** Smooth browsing experience for various yogurt flavors and toppings.
* 📝 **User Authentication:** Pre-built forms for user registration and login.
* ⚡ **Fast Loading:** Built entirely with lightweight static files (No heavy frameworks).

---

## 🛠️ Tech Stack & Tools

* **HTML5:** Semantic structure and content organization.
* **CSS3:** Custom layout styling, Flexbox/Grid, and hover effects.
* **JavaScript (Vanilla):** Mobile menu toggles, interactive elements, and basic form handling.
* **GitHub Pages:** Free and reliable static web hosting.

## 🚀 How to Run Locally

1. Clone this repository:
   ```bash
   git clone https://github.com
   ```
2. Navigate to the project folder:
   ```bash
   cd Web-Lab-Exercises
   ```
3. Open `index.html` in any web browser.
